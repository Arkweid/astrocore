# Contributing
Thanks for taking the time to contribute! Feel free to make Pull Request with you changes.

# Requirements

Rust >= 1.33.0

sodiumoxide libs in your OS

# Proccess

1. Fork repo
2. git clone <forked_repo>
3. run commands:
```bash
RUSTFLAGS=-Awarnings cargo run
```
4. Make changes
5. Push your changes in repo
